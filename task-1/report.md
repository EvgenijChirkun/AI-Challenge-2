# Task 1 Report

## Approach

The leaderboard was recreated as a static React + TypeScript application. The interface includes a filter bar, progressive participant search, a top-3 podium, a ranked participant list, category indicators, total score summaries, and expandable activity details for each participant.

## Tools and Techniques

- VSCode
- GitHub Copilot
- Vite
- React
- TypeScript
- CSS
- GitHub Pages

## Data Replacement

All participants, roles, avatars, activities, dates, and scores were replaced with synthetic mock data created from scratch. Real photos were not used; the UI uses initials-based placeholder avatars with generated gradients.

## Responsible AI Usage

No real corporate data, real names, real photos, real departments, real job titles, or personal data were provided to AI tools or included in this repository.
